# Creación compoñentes personalizados con propiedades complexas

# Pasos

Para crear un compoñente personalizado con propiedades complexas faremos o seguinte :

# Creación de clase co nome do compoñente

Faremos o mesmo ca no anterior caso, crearemos unha clasecon constructor sen argumentos, setter e getters e implements Serializable.

public class BotonConPropiedadesComplexas extends JButton implements Serializable {
    
    private CoresBoton coresBoton;
    
    public BotonConPropiedadesComplexas() {
        
    }
    
    public CoresBoton getCoresBoton() {
        return coresBoton;
    }
    
    public void setCoresBoton(CoresBoton coresBoton) {
        this.coresBoton = coresBoton;
        
        setBackground(coresBoton.getColorFondo());
        setForeground(coresBoton.getColorTexto());
    }
    
}

# Creación da clase (atributoComplexo)

Crearemos esta clase engadindo as propiedades que nos interesen engadir.

public class CoresBoton {

    private Color colorFondo;
    private Color colorTexto;

    public CoresBoton(Color colorFondo, Color colorTexto) {
        this.colorFondo = colorFondo;
        this.colorTexto = colorTexto;
    }

    public Color getColorFondo() {
        return colorFondo;
    }

    public void setColorFondo(Color colorFondo) {
        this.colorFondo = colorFondo;
    }

    public Color getColorTexto() {
        return colorTexto;
    }

    public void setColorTexto(Color colorTexto) {
        this.colorTexto = colorTexto;
    }

}

# Creación da clase (nomePropiedade)PropertyEditorSupport,

Crearemos dita clase que deberá extender de PropertyEditorSupport, deberás implementar os métodos (supportCustomEditor), (getCustomEditor, getJavaInitializationString), (getValue).

public class CoresBotonPropertyEditorSupport extends PropertyEditorSupport {

    private CoresBotonPanel coresBotonPanel = new CoresBotonPanel();

    @Override
    public boolean supportsCustomEditor() {
        return true;
    }

    @Override
    public Component getCustomEditor() {
        return coresBotonPanel;
    }

    @Override
    public String getJavaInitializationString() {
        CoresBoton coresBoton = coresBotonPanel.getSelectedValue();
        return "new com.mycompany.ejercicio2gui.CoresBoton(" + "new java.awt.Color("
                + coresBoton.getColorFondo() + "), " + coresBoton.getColorTexto() + ")";
    }

    @Override
    public Object getValue() {
        return coresBotonPanel.getSelectedValue();
    }

}
